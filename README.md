# 🚀 Todo Pro Web App

A professional, full-featured To-Do List Web Application built using:

- HTML, CSS, JavaScript (Frontend)
- Google Sheets (Database)
- Google Apps Script (Backend API)

---

## ✨ Features

### 🔐 Authentication

- Register & Login system
- OTP verification (simulated)
- Form validation

### 🎨 UI/UX

- Glassmorphism design
- Dark/Light mode
- Responsive layout
- Smooth animations

### 📁 Task Management

- Multiple lists
- Task creation & editing
- Status tracking:
  - Not Started
  - In Progress
  - Completed
  - Delayed

### 🔎 Filtering & Search

- Filter by status
- Real-time search

### 🗃 Data System

- Google Sheets database
- Archive instead of delete

---

## 🧠 Tech Architecture

Frontend → JavaScript Fetch API → Google Apps Script → Google Sheets

2. Create Google Sheet database

4. Paste API URL in `api.js`
5. Open `index.html`
